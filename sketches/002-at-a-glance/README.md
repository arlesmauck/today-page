## Variant: At-a-Glance Dashboard

### Design stance
An instrument panel for your day. Everything visible in a single scan, no scrolling required. Dark mode, high contrast, information-dense but organized. Designed for the "I'm running out the door" moment.

### Key choices
- **Layout:** Three-column CSS grid on desktop. Left = weather + schedule + stats. Center = scrollable news feed with filters. Right = Reddit + tomorrow preview + alerts.
- **Typography:** Inter throughout. Tabular nums for numbers. Tight line-height for scan-ability.
- **Color:** Near-black background (#0c0c0e) with zinc grays. Colored pills for categories. Purposefully utilitarian.
- **Interaction:** Category filters on news. Hover states. Live clock. Expandable/hoverable detail cards.

### Trade-offs
- **Strong at:** Pure efficiency. You can ingest everything in under 60 seconds.
- **Weak at:** Can still feel like "information overload" if you're already stressed. The density is the point, but it's not calming.

### Best for
- Power users who want data, not poetry.
- Busy mornings when you need situational awareness fast.
